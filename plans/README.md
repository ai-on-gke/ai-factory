# Plans

This directory contains execution plans for spec-driven development in the `ai-factory` project.
Each plan is stored in a subdirectory named with the current date and a descriptive name (e.g., `YYYY-MM-DD_plan-name`).

## Structure

Inside each plan directory, you will find:

- `plan.yaml`: The main plan file containing a list of tasks and dependencies.
- Auxiliary files (optional): Markdown files named `task-name.md` providing specific guidance for individual tasks.

## Validation

Plans can be validated using the `tool plan validate` command.
